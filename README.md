# 🥩 MeetStore

MeetStore is a modern online meat delivery web application that allows customers to order fresh meat online with a simple and user-friendly interface. The platform includes a customer portal, an admin dashboard, secure authentication, order management, and online payment integration.

## 🌐 Live Demo

🔗 https://meetstore.netlify.app

## 📂 GitHub Repository

🔗 https://github.com/makizhnans-cell/meetstore

## ✨ Features

- 🔐 User Authentication (Email/Phone)
- 🛒 Add to Cart
- 📦 Order Tracking
- 📱 Mobile Responsive Design
- 👨‍💼 Admin Dashboard
- 🛍️ Product Management
- 📋 Order Management
- 🔥 Firebase Integration
- ⚡ Fast and Modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Firebase
- Netlify (Hosting)
- Git & GitHub

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/makizhnans-cell/meetstore.git
```

2. Open the project folder.

3. Configure Firebase credentials.

4. Open `index.html` in your browser or deploy using Netlify
